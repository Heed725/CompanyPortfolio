# Emnet Company Portfolio

Blue-themed static portfolio website for Emnet, an IT company based in Dar es Salaam, Tanzania.

## Stack

- HTML
- CSS
- JavaScript
- GitHub Pages via GitHub Actions

## Local preview

Open `index.html` in a browser.

## Deployment

Pushes to `main` trigger the workflow in `.github/workflows/pages.yml`, which deploys the site to GitHub Pages.
